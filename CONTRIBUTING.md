# Contributing

This repository is maintained as a professional Android portfolio project.

## Quality Bar

Before submitting changes:

- Keep Android screens and data models organized.
- Do not commit local secrets or signing files.
- Update README instructions when setup changes.
- Add tests for role-sensitive flows where practical.
- Preserve clean UI behavior across screen sizes.

## Pull Request Checklist

- Summary of the change
- Testing notes
- Screenshots for UI changes
- Data model notes, if relevant
- Known limitations
